# Security

Anchor handles journal-like data, so privacy matters.

## Reporting Issues

Please do not open a public issue with private data, credentials, or a reproducible privacy exposure that includes someone else's content.

Instead, contact the project maintainer privately if a security or privacy issue needs disclosure.

## Data Safety

This public repo should not contain real journal entries, private archive exports, or personal media. Use a separate private folder or repository for live Anchor data.
